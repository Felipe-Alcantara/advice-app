# advice-app

It's a Javascript application that shows advices

## Styling changes

- Justifiquei o texto do conselho adicionando `text-align: justify` em `.advice-quote p`.
- Adicionei `hyphens: auto` e prefixes para melhorar a hifenização em navegadores que suportam.
- Também apliquei as mesmas regras em `.original-text` para consistência quando o texto original está visível.

Se quiser que a última linha fique alinhada de outra forma (por exemplo, à esquerda) remova `text-align-last: center`.
